* # Ejercicio 1- Automatización E2E
* Previo a la ejecución se debe tener instalado Cypress
* Descargar proyecto de GitHub https://github.com/Shomira/pruebaDevsu.git
* Abrir un cmd en la ubicación del poryecto
* Colocar el comando npx cpress open
* En la interfaz de Cypress seleccionar E2E Testing
* Elegir navegador Chrome
* Ir al modulo de Specs y seleccionar el archivo ejercicio1.cy o ejercicio2.cy
